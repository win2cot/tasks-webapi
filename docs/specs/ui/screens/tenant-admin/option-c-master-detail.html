強み
一覧と詳細を常時同時表示。左で次々選択するだけで右に S-14 が即時差し替わり、複数テナントを連続して確認・状態切替するパワーユーザの操作効率が最も高い。一覧の文脈を失わずに 1 件を深掘りでき、選択行はハイライトで追跡できる。

想定読者
多数テナントを横断して棚卸し・一括点検する運用パワーユーザ。「次々に切り替えて状態を確認」する頻度が高い PC 中心の専任 SaaS Admin。
