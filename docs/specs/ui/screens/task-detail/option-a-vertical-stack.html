タスク詳細 #152

Tenant Admin でも業務タスクへの特権はありません(ADR-0005)。本画面の操作可否は 所有者 / 担当者 / 関係者 の3役割評価のみで決まります。

設計判断ポイント — 案A 縦スクロール型(Vertical Stack)

強み
1カラム縦積みで上から「基本情報 → メタ → 関係者 → 監査メタ → 操作」と読み下せる。S-03 ダッシュボードのセクションカード哲学をそのまま継承し、タブレット縦・狭幅でも崩れない。実装が最も素直。
弱み
PC の横幅を使い切れず、メタ項目を見るのにスクロールが要る。説明とメタを同時に見比べづらく、情報密度は3案中もっとも低い。
想定読者
1件のタスクを腰を据えて読む Member。タブレット併用や通知メールリンクからの単票閲覧が多いユーザー。
